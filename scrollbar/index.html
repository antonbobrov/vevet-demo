ScrollBar

Docs / Source Code

A custom scroll bar component that can be added to custom or native scroll containers. It supports both horizontal and vertical scroll bars, auto-hide functionality, and drag-to-scroll.

Requires styles. See the source code.

The scrollbar is applied to the window. Look at the right side of the page.
